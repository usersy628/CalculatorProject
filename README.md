# 🧮 Calculator Project

## 📌 프로젝트 소개

Java를 활용하여 구현한 간단한 콘솔 기반 계산기 프로그램입니다.
사칙연산 기능을 중심으로, 계산 결과 저장 및 조회 기능을 포함하고 있습니다.

객체지향 설계를 기반으로 클래스 간 역할을 분리하여 구조적으로 구현하였습니다.
기본적인 계산기 기능을 넘어,
객체지향 설계와 Java 문법을 실제로 적용해볼 수 있었던 프로젝트입니다.

---

## 🚀 주요 기능

* 사칙연산 수행 (`+`, `-`, `*`, `/`)
* 계산 결과 저장
* 저장된 결과 목록 조회
* 특정 값 이상 결과 조회
* 가장 먼저 저장된 결과 삭제
* 메뉴 선택 기반 콘솔 인터페이스
* 잘못된 입력 및 예외 처리

---

## 🧱 프로젝트 구조

```
CalculatorProject
 ├── App.java                  // 프로그램 실행 및 전체 흐름 제어
 ├── Calculator.java           // 결과 저장 및 관리
 ├── ArithmeticCalculator.java // 실제 계산 수행
 └── Operator.java             // 연산자 enum 및 연산 정의
```

---

## ⚙️ 설계 특징

### 1. 역할 분리 (Separation of Concerns)

* **App**: 사용자 입력 및 프로그램 흐름 제어
* **Calculator**: 계산 결과 저장 및 조회
* **ArithmeticCalculator**: 실제 계산 수행
* **Operator**: 연산 규칙 정의

---

### 2. Enum 활용

연산자를 `enum`으로 정의하여 switch문 없이도 안전한 연산 처리가 가능합니다.

---

### 3. 제네릭(Generic) 활용

`ArithmeticCalculator<T extends Number>` 구조를 통해 다양한 숫자 타입을 처리할 수 있도록 설계되었습니다.

---

### 4. Stream API 활용

저장된 결과 중 특정 조건을 만족하는 값을 조회할 때 Stream을 사용하여 구현하였습니다.
